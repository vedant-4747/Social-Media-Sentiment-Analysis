# 🗣️ PRODIGY_DS_04: Social Media Sentiment Analysis

This repository contains my submission for **Task 03** of the Prodigy InfoTech Data Science Internship.

## 📄 Project Overview
**Objective:** Analyze and visualize sentiment patterns in social media data to understand public opinion.
**Dataset:** [Twitter Sentiment Analysis Dataset](https://www.kaggle.com/datasets/jp71213/twitter-entity-sentiment-analysis).

I analyzed tweet data to classify public sentiment as **Positive**, **Negative**, or **Neutral**. As a blockchain enthusiast, I view this as a foundational step in building **"Crypto Fear & Greed"** models—using social signals (like "Crypto Twitter") to predict market volatility and token price action.

## 📊 Key Findings
* **Brand Sentiment:** visualized how specific brands or topics trigger polarized reactions.
* **Negative Bias:** Identified that negative sentiment often propagates faster than positive sentiment (a key metric for crisis management).
* **Data Visualization:** Used word clouds and bar charts to display the most frequent terms associated with each sentiment class.

## 🛠 Technologies Used
* **Python** (Pandas, NumPy)
* **Natural Language Processing (NLP)** (NLTK / TextBlob / Scikit-learn)
* **Matplotlib & Seaborn** (For visualization)
